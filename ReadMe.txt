 Hello, 大家好

我们每个人都会开发各种各样的应用, 但是我们一般都会用到一些常用的库, 有一些库封装的很好, 我们用起来很舒服, 但是也有一些库封装的不好, 我们用起来很辛苦...
所以,我想创建一个通用的框架...

我已经把代码开源到GitHub上了, 也希望大家可以贡献自己的能力, 然后完善这个框架, 谢谢大家了..

简单的使用说明:
1, 创建一个新的工程,(由于Framework中封装了很多开源的库, 所以只建议新工程使用)
2, 导入下载好的GeneralFramework文件件到自己的工程中
3, 在创建ViewController的时候, 继承CommandController
4, 因为已经封装好一些方法, 调用的时候直接:[self 方法名]就行
5, 希望大家也可以奉献自己的想法和行动, 共同让开源项目更好的发展..
